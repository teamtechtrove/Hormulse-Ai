# 🚀 Hormulse AI - The AI Assistant Built For Everyone

**Powered by 6 AI Providers — All Free**

Hormulse AI is a modern, user-friendly application that gives you access to multiple state-of-the-art AI models in one place. Chat with Google Gemini, OpenAI GPT-4, Anthropic Claude, Cohere, DeepSeek, and more — all for free.

## ✨ Features

- 🤖 **Multi-AI Support** - Access 6 different AI providers
- 💬 **Real-time Chat** - Instant responses from your chosen AI
- 🖼️ **Image Analysis** - Upload and analyze images with AI
- 💾 **Chat History** - Automatically save your conversations
- ⚡ **Lightning Fast** - Optimized for speed and performance
- 🔒 **Secure & Private** - Your data is encrypted
- 🌐 **Responsive Design** - Works on desktop and mobile
- 🎨 **Modern UI** - Beautiful, intuitive interface

## 🛠️ Tech Stack

- **Backend**: Python, Flask
- **Frontend**: HTML5, CSS3, JavaScript
- **Database**: SQLite / PostgreSQL
- **AI APIs**: 
  - Google Gemini
  - OpenAI GPT-4
  - Anthropic Claude
  - Cohere
  - DeepSeek
  - And more

## 📋 Prerequisites

- Python 3.8 or higher
- pip (Python package manager)
- Virtual environment (recommended)
- API keys for AI providers (free tier available for most)
